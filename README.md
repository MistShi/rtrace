# Rtrace

Traceroute subset with geocoding using the geocoder gem. Tested on OSX only.

## Installation

Add this line to your application's Gemfile:

    gem 'rtrace'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rtrace

## Usage

Just do rtrace -h
